# EmotionGame

## 简介

这是一个测试默契度的小游戏

## 游戏规则

屏幕上随机出现图片或文字，根据提示，参与游戏的两个人需要做出相应的表情。

根据两人表情的相似程度，打分越高，默契程度越好。
